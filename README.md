# 🎨 Guess the Color

A fun multiplayer color-pattern guessing game. Crack the hidden bottle pattern before your rivals do!

## ▶️ Play Now

**[👉 Play the game here](https://sadib400.github.io/color-code-game/)**

No install needed — works in any modern browser, on desktop and mobile.

## How to Play

A hidden color pattern sits in the bottom row, built from the exact bottles you see on the racks — every color is used only once (no repeats). On your turn, place bottles on **2 slots** in the top row and submit:

- **0 correct** → next player's turn.
- **1 of 2 correct** → you must point out which one is right. Guess right and it locks in (+1 point) and you keep playing. Guess wrong and your turn ends.
- **Both correct** → both lock instantly (+2 points) and you keep playing.

There are **5 levels** and the board grows each level (3 → 7 slots). Whoever locks in the most slots wins the trophy! 🏆

## Game Modes

- **📱 Same Device** — 2–4 players pass-and-play on one screen. Any seat can be switched to a 🤖 Computer player (so you can play solo vs CPU).
- **🌐 Online** — host creates a room and gets a 4-letter code; friends join from their own devices with the code. Powered by peer-to-peer WebRTC (PeerJS), no account needed.

## Features

- Optional player names
- Smart CPU opponents that deduce from public clues
- Light, playful, animated UI with confetti celebrations
- Big animated winner leaderboard
